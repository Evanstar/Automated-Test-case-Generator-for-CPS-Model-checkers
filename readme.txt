Requirements: OS: any linux distribution.


The Java Application can be run in the following steps:


1. Open the Project folder in eclipse.

2. Right click on the folder and click on "Run As" -> Java Application

3. Enter the template required for the test-cases and click on generate.

4. For the C++ file :

	1. Go to Spaceex folder and compile the the C++ file giving the file path as follows 
		"g++ [file_path]/scrip.cpp"
	2. Next, run the executable file as :
		"./a.out"
